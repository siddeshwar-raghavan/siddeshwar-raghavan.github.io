Patents
Human Interface System For Playing Virtual Percussion Instruments
Patent published at Indian Patent Office. ID no. IN 201841021574
Siddeshwar Raghavan, Aravind Shankara Narayanan

This is a virtual reality learning environment for percussion instruments. It provides haptic feedback to the users based on the force of strike of the percussion instrument. Consisting of in-house developed sensor array for the detection of position of the hand and coordinating with a smartphone containg the application developed. The experience can be enjoyed through a Google Cardboard. The prototype developed contains various learning modes, floating notes, highlighting the drum to be struck and a reverse karaoke mode (Music to be played, lyrics run in the background) to help the users learn.

This patent was published in April 2019.
